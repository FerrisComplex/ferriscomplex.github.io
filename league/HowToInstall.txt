Prerequisites
 - EnsoulSharp installed from http://ensoulsharp.com/

Installing Local
  Download FerrisAIO.esplugin to any folder
  Click Install new assembly on EnsoulSharp
  Click Local field and select the folder FerrisAIO.esplugin is in (It will show an empty folder when you click it)
  Click add and your good to go
  Gif example https://gyazo.com/2f02ea8dcf3a7e44d93abc82305c1d54


Installing from Github
  Install new assembly and type the link bellow in github to install
  https://github.com/FerrisComplex/ferriscomplex.github.io
  Gif example https://gyazo.com/ac4eb1e30ce2f721f750dd88baa086f4
